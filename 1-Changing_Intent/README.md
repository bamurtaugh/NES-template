# 1-Changing_Intent

This folder contains examples demonstrating how to modify and change the intent of existing code with AI assistance.

## Examples

### TypeScript - Geometry (geometry.ts)
A Point class demonstrating basic geometric calculations and distance computations.

### C++ - Statistics (stats.cpp)
A Statistics class that provides statistical calculations including mean, standard deviation, and minimum values.

### JavaScript - Calculator (calculator.js)
A simple calculator implementation showing basic arithmetic operations and function modifications.

### C# - Data Processor (DataProcessor.cs)
A data processing utility demonstrating object-oriented programming patterns and data manipulation.

## Usage

These examples are designed to showcase how AI can help you:
- Modify existing function behavior
- Change class implementations
- Refactor code intent while maintaining functionality
- Add new methods to existing classes

Try experimenting with these files using GitHub Copilot to see how it suggests modifications and improvements to the existing code structure.
