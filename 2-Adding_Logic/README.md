# Adding Logic Examples

This folder contains examples that demonstrate how GitHub Copilot's Next Edit Suggestions (NES) can help when adding new functionality and logic to existing codebases.

## Examples

### game.py  
A Python two-player guessing game that showcases how NES can suggest additions for:
- New game features and rules
- Enhanced player interaction
- Additional validation logic
- Extended game mechanics

**Try this scenario:**
- Add difficulty levels with different number ranges
- Implement a scoring system
- Add hints or clues functionality
- Create a tournament mode for multiple rounds

### extension-sample/
A VS Code extension example that demonstrates how NES can help when extending existing functionality:
- Adding new commands and features
- Extending the VS Code API usage
- Adding configuration options
- Implementing additional UI components

## About This Category

These examples focus on scenarios where you're building upon existing code by adding new features, logic, or capabilities. NES excels at understanding the existing patterns and suggesting consistent additions that fit well with the current codebase structure and style.
