# NES Template Repository

This repository contains example scenarios for demonstrating GitHub Copilot's Next Edit Suggestions (NES) feature. Each numbered folder represents a different category of editing scenarios that showcase how NES can help developers write better code more efficiently.

## Repository Structure

- **1-Changing_Intent** - Examples of changing code purpose and functionality
- **2-Adding_Logic** - Scenarios for adding new features and logic to existing code  
- **3-Changing_Fixing_Logic** - Examples of modifying and fixing existing logic
- **4-Refactoring** - Code restructuring and improvement scenarios
- **5-Fuller** - Comprehensive, full-featured application examples

## Getting Started

Each folder contains specific examples and instructions for trying out different NES scenarios. Navigate to any folder to explore the examples and follow the README instructions for hands-on experience with Copilot's editing suggestions.

## About NES

Next Edit Suggestions (NES) is a feature of GitHub Copilot that provides contextual suggestions for your next edit based on the changes you're making and the surrounding code context. These examples help you understand how to effectively use NES in various development scenarios.
