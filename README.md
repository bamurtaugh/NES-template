# NES Template Repository

This repository contains examples and templates for demonstrating **GitHub Copilot's Next Edit Suggestions (NES)** feature. NES helps developers by suggesting contextual code changes and improvements as they write and edit code.

## üöÄ Quick Start

1. Clone this repository
2. Browse the example directories (1-5) for different NES demonstration scenarios
3. Use the examples with GitHub Copilot enabled to see NES in action
4. Follow the instructions in each example's documentation

## üìÅ Repository Structure

This repository is organized into 5 main categories of examples:

| Directory | Purpose | Description |
|-----------|---------|-------------|
| [`1-Changing_Intent/`](./1-Changing_Intent/) | Changing Code Intent | Examples of modifying existing code to change its purpose or behavior |
| [`2-Adding_Logic/`](./2-Adding_Logic/) | Adding New Logic | Examples of adding new functionality to existing code |
| [`3-Changing_Fixing_Logic/`](./3-Changing_Fixing_Logic/) | Changing & Fixing Logic | Examples of modifying and fixing existing logic |
| [`4-Refactoring/`](./4-Refactoring/) | Code Refactoring | Examples of restructuring and improving existing code |
| [`5-Fuller/`](./5-Fuller/) | Complete Example | A full-featured task manager app demonstrating advanced NES scenarios |
